rule sets

selector {
  property:value;
  property:value;
  property:value;
}
  • each property:value is called a declaration

selectors

element

E.g. all <p> elements

p {
  text-align: center;
  color: red;
}

E.g. all <h1>, <h2> and <p> elements

h1, h2, p {
  text-align: center;
  color: red;
}

id

E.g. the element with id="para1"

#para1 {
  text-align: center;
  color: red;
}

class

E.g. all elements with class="center"

.center {
  text-align: center;
  color: red;
}

E.g. all <p> elements with class="center"

p.center {
  text-align: center;
  color: red;
}

inserting CSS

external style sheet

defined within the <link> element inside the <head> section

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

where the mystyle.css file looks like

body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}

internal style sheet

defined within the <style> element inside the <head> section

<head>
<style>
body {
  background-color: linen;
}

h1 {
  color: maroon;
  margin-left: 40px;
} 
</style>
</head>

inline style

defined within the style attribute of a single element

<h1 style="color:blue;margin-left:30px;">This is a heading</h1>

units

  • https://www.w3schools.com/css/css_units.asp

absolute

  • units are fixed - i.e. a length expressed in any of these will appear as exactly that size
  • not recommended for use on screen but useful for print layout
Unit Description
cm centimeters
mm millimeters
in inches (1in = 96px = 2.54cm)
px pixels (1px = 1/96th of 1in)
pt points (1pt = 1/72 of 1in)

relative

  • specify length relative to another length property
  • scales better between different rendering mediums
Unit Relative to
em the font-size of the element (2em = twice the size of the current font)
ex the x-height of the current font (rarely used)
ch width of the “0” (zero)
rem font-size of the root element
vw 1% of the width of the viewport (i.e browser window)
vh 1% of the height of the viewport (i.e browser window)
vmin 1% of viewport’s smaller dimension
vmax 1% of viewport’s larger dimension

examples

font size

good generic approach that works in all browsers is to set the default size in percent for the body element and then use em for other text elements.

body {
  font-size: 100%;
}

h1 {
  font-size: 2.5em;
}

h2 {
  font-size: 1.875em;
}

p {
  font-size: 0.875em;
}
