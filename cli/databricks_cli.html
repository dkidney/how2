authentication

  • https://docs.databricks.com/dev-tools/api/latest/authentication.html#authentication

Get a personal access token from the databricks UI:

Account >> User Settings >> Access Tokens >> Generate New Token >> Generate

Copy the generated token and store in a secure location.

To configure the CLI to use the access token, run the following and follow the prompts

databricks configure --token

After you complete the prompts, your access credentials are stored in the file ~/.databrickscfg.
