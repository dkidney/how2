github pages

https://pages.github.com/

  • sign in to Github and navigate to your repo
  • go to Settings
  • scroll down to the Github Pages section
  • under Source select master branch (or whatever option you prefer)
  • choose a theme if you want one
  • your pages should now be available at http://username.github.io/repository
