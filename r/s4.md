## Create an instance

### `dgCMatrix`

Create a normal matrix.

``` r
mat = matrix(c(1,0,0,0,1,0,0,0,1,0,1,0), 4, 3, byrow = TRUE)
mat
```

    ##      [,1] [,2] [,3]
    ## [1,]    1    0    0
    ## [2,]    0    1    0
    ## [3,]    0    0    1
    ## [4,]    0    1    0

Convert to `dgCMatrix`.

``` r
index = which(mat == 1, arr.ind = TRUE)
smat = Matrix::sparseMatrix(
  i = index[,1], 
  j = index[,2], 
  x = 1,
  dims = dim(mat),
  dimnames = dimnames(mat)
)
smat
```

    ## 4 x 3 sparse Matrix of class "dgCMatrix"
    ##           
    ## [1,] 1 . .
    ## [2,] . 1 .
    ## [3,] . . 1
    ## [4,] . 1 .

``` r
str(smat)
```

    ## Formal class 'dgCMatrix' [package "Matrix"] with 6 slots
    ##   ..@ i       : int [1:4] 0 1 3 2
    ##   ..@ p       : int [1:4] 0 1 3 4
    ##   ..@ Dim     : int [1:2] 4 3
    ##   ..@ Dimnames:List of 2
    ##   .. ..$ : NULL
    ##   .. ..$ : NULL
    ##   ..@ x       : num [1:4] 1 1 1 1
    ##   ..@ factors : list()

Create the same object from scratch.

``` r
smat2 = new(
  Class = "dgTMatrix",
  i = index[,1] - 1L,
  j = index[,2] - 1L,
  Dim = dim(mat),
  Dimnames = list(NULL, NULL),
  x = rep(1, nrow(index)),
  factors = list()
)
smat2 = as(smat2, "dgCMatrix")
identical(smat, smat2)
```

    ## [1] TRUE
